NEXT_PUBLIC_SUPABASE_URL - from supabase.com → New Project → Settings → API

ANTHROPIC_API_KEY - from console.anthropic.com → API Keys → Create Key
API_SPORTS_KEY - from api-sports.io → Dashboard → My Account → API Key
ODDS_API_KEY - from the-odds-api.com → Account → API Key
NEWS_API_KEY - from newsapi.org → Account → API Key
WEATHER_API_KEY - from weatherapi.com → Dashboard → API Key
STRIPE_SECRET_KEY - from dashboard.stripe.com → Developers → API Keys → Secret key
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY - from dashboard.stripe.com → Developers → API Keys → Publishable key
STRIPE_WEBHOOK_SECRET - from dashboard.stripe.com → Webhooks → Add endpoint → Signing secret
STRIPE_PRO_PRICE_ID - from dashboard.stripe.com → Products → Create product "PrediQs Pro $14.99/mo" → copy Price ID
STRIPE_ELITE_PRICE_ID - from dashboard.stripe.com → Products → Create product "PrediQs Elite $39.99/mo" → copy Price ID
ONESIGNAL_APP_ID - from onesignal.com → New App → Settings → Keys & IDs
ONESIGNAL_API_KEY - from onesignal.com → Settings → Keys & IDs → REST API Key
RESEND_API_KEY - from resend.com → API Keys → Create API Key
NEXT_PUBLIC_URL - your Replit app URL e.g. https://prediqs-ai.yourusername.repl.co
