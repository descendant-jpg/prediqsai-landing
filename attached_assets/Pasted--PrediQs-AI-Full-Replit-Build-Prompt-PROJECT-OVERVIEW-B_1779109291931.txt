# PrediQs AI — Full Replit Build Prompt

---

## PROJECT OVERVIEW

Build **PrediQs AI** — a full-stack AI-powered sports betting intelligence platform. This is NOT a sportsbook. It is a prediction, analysis, and decision intelligence tool for sports bettors covering **NFL, NBA, MLB, and Soccer**.

The platform has three subscription tiers: **Free, Pro ($14.99/mo), and Elite ($39.99/mo)**.

---

## TECH STACK

- **Frontend:** Next.js 14 (App Router) + Tailwind CSS
- **Backend:** Next.js API Routes (serverless)
- **Database & Auth:** Supabase (PostgreSQL + Auth)
- **AI:** Anthropic Claude API (`claude-sonnet-4-20250514`)
- **Sports Data:** API-Sports (api-sports.io) — NFL, NBA, MLB, Soccer
- **Odds Data:** The Odds API (the-odds-api.com)
- **News:** NewsAPI (newsapi.org)
- **Weather:** WeatherAPI (weatherapi.com)
- **Payments:** Stripe (subscriptions)
- **Notifications:** OneSignal (push notifications)
- **Email:** Resend
- **Deployment:** Replit (dev) → Vercel (prod)

## PREDICTION ENGINE LOGIC

Create `/api/predictions/generate` that runs daily at 8am:

```
1. Fetch today's games (NFL + NBA + MLB + Soccer) from API-Sports
2. For each game:
   a. Get current odds from The Odds API
   b. Calculate bookmaker implied probability from odds
   c. Get recent team form (last 5 games)
   d. Get injury reports
   e. Get weather for outdoor stadiums
   f. Get latest news mentions for both teams
3. Send all data to Claude API → get structured JSON prediction
4. Calculate value: if ai_probability > bookmaker_probability + 5 → value_detected = true
5. Save prediction to Supabase predictions table
6. Mark tier_required:
   - free: first 3 picks of day, confidence shown
   - pro: all picks + reasoning + key factors
   - elite: all picks + all data + live updates
```

## TIER GATING LOGIC

```javascript
// lib/tierCheck.ts
export type Tier = 'free' | 'pro' | 'elite';

export function canAccess(userTier: Tier, requiredTier: Tier): boolean {
  const levels: Record<Tier, number> = { free: 0, pro: 1, elite: 2 };
  return levels[userTier] >= levels[requiredTier];
}

// In API routes:
const { data: profile } = await supabase
  .from('profiles')
  .select('tier')
  .eq('id', userId)
  .single();

if (!canAccess(profile.tier, 'pro')) {
  return Response.json({
    error: 'upgrade_required',
    message: 'Upgrade to PrediQs Pro to access this feature',
    upgrade_url: '/pricing'
  }, { status: 403 });
}
```

**UI tier gate component:**
```jsx
// components/ui/TierGate.tsx
export function TierGate({ requiredTier, userTier, children }) {
  if (canAccess(userTier, requiredTier)) return children;
  return (
    <div style={{ position: 'relative' }}>
      <div style={{ filter: 'blur(6px)', pointerEvents: 'none' }}>{children}</div>
      <div className="absolute inset-0 flex items-center justify-center bg-black/50 rounded-lg">
        <div className="text-center p-4">
          <div className="text-yellow-400 font-bold mb-2">
            {requiredTier === 'pro' ? '⚡ PRO Feature' : '👑 ELITE Feature'}
          </div>
          <a href="/pricing" className="bg-cyan-500 text-black px-4 py-2 rounded font-bold text-sm">
            Upgrade Now
          </a>
        </div>
      </div>
    </div>
  );
}
```

## STRIPE SUBSCRIPTION SETUP

```javascript
// api/stripe/create-checkout/route.ts
import Stripe from 'stripe';
const stripe = new Stripe(process.env.STRIPE_SECRET_KEY);

export async function POST(req) {
  const { priceId, userId, email } = await req.json();
  
  const session = await stripe.checkout.sessions.create({
    mode: 'subscription',
    payment_method_types: ['card'],
    line_items: [{ price: priceId, quantity: 1 }],
    success_url: `${process.env.NEXT_PUBLIC_URL}/dashboard?upgraded=true`,
    cancel_url: `${process.env.NEXT_PUBLIC_URL}/pricing`,
    customer_email: email,
    metadata: { userId },
  });
  
  return Response.json({ url: session.url });
}

// api/stripe/webhook/route.ts
// On checkout.session.completed → update profiles.tier = 'pro' or 'elite'
// On customer.subscription.deleted → update profiles.tier = 'free'
```

### Confidence Arc Component
```jsx
// components/ui/ConfidenceMeter.tsx
export function ConfidenceMeter({ value }: { value: number }) {
  const color = value >= 70 ? '#00FF94' : value >= 50 ? '#FFD700' : '#FF4D4D';
  const circumference = 2 * Math.PI * 40;
  const offset = circumference - (value / 100) * circumference;
  
  return (
    <div className="relative w-24 h-24 flex items-center justify-center">
      <svg className="absolute" width="96" height="96" style={{ transform: 'rotate(-90deg)' }}>
        <circle cx="48" cy="48" r="40" fill="none" stroke="#131E2E" strokeWidth="6" />
        <circle cx="48" cy="48" r="40" fill="none" stroke={color}
          strokeWidth="6" strokeDasharray={circumference}
          strokeDashoffset={offset} strokeLinecap="round"
          style={{ transition: 'stroke-dashoffset 1s ease' }} />
      </svg>
      <div style={{ color, fontFamily: 'DM Mono', fontWeight: 700, fontSize: '20px' }}>
        {value}%
      </div>
    </div>
  );
}
```
