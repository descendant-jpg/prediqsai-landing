     Text: "Past performance of any trading signal or strategy does 
     not guarantee future results. Historical data is provided for

  4. "Your Responsibility" (teal/green heading)
     Text: "You are solely responsible for your trading decisions.

  • We are NOT a registered investment advisor
  • We are NOT a licensed broker or financial institution
  • Signals are for educational purposes only
  • Always do your own research before trading
  • Never risk more than you can afford to lose

- Fixed bottom button (gold/amber): "I Understand and Accept"
  (saves acceptance to localStorage, never shows again)

DESIGN SYSTEM:
- Background: #0d0d0d (near black)
- Card background: #141414 with rgba(255,255,255,0.08) border
- Primary accent: #F5A623 (gold/amber)
- Secondary accent: #00C9A7 (teal)
- Danger/warning: #E53935 (red)
- Text primary: #ffffff
- Text secondary: #999999
- Border radius on cards: 20px
- Border radius on buttons: 14px
- Font: DM Sans (Google Fonts) — 400, 700, 800 weights
- Buttons have subtle box-shadow and scale(1.02) on hover
- Modals animate in with a slide-up + spring effect
- All modals have backdrop blur overlay: rgba(0,0,0,0.75) + blur(6px)

---

TECH STACK:
- React (functional components + hooks)
- No external UI libraries (pure CSS-in-JS with inline styles or 
  CSS modules)
- Import DM Sans from Google Fonts in index.html
- Use localStorage to persist: accepted disclaimer, selected language
- Mobile-first layout, max modal width 380px, centered on desktop
