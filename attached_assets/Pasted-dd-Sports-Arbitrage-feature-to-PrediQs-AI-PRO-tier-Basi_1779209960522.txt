HOW ARBITRAGE WORKS IN THE APP:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Fetch odds from multiple bookmakers
   via The Odds API (ODDS_API_KEY)

2. For each game compare odds across
   all available bookmakers

3. Calculate if arbitrage exists:

   For 2-way markets (home/away):
   arb exists if:
   (1/odds1) + (1/odds2) < 1.0

   For 3-way markets (home/draw/away):
   arb exists if:
   (1/odds1) + (1/odds2) + (1/odds3) < 1.0

   Profit % = (1 - sum) × 100

4. If arb detected → alert user instantly
